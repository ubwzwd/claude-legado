    Create `src/novel/display.py` per D-04, D-05, D-06, D-10, D-11, D-12:

    **Constants:**
    ```python
    PAUSE_LONG = 0.15    # sentence-end: 。！？
    PAUSE_MED = 0.06     # clause-end: ，、；
    DELAY_MIN = 0.015    # 15ms base min
    DELAY_MAX = 0.040    # 40ms base max
    BURST_PROB = 0.15    # 15% chance of burst
    BURST_MIN = 8
    BURST_MAX = 15
    ```

    **Functions:**

    `_char_delay(ch: str) -> float`:
    - If ch in '。！？': return PAUSE_LONG
    - If ch in '，、；': return PAUSE_MED
    - Otherwise: return random.uniform(DELAY_MIN, DELAY_MAX)

    `stream_text(text: str) -> None`:
    - Loop through text character-by-character using index `i`.
    - At each position, with probability BURST_PROB and if enough chars remain (i + BURST_MIN < len(text)), emit a burst: write text[i:i+chunk_size] where chunk_size = random.randint(BURST_MIN, BURST_MAX), flush, sleep random.uniform(DELAY_MIN, DELAY_MAX), advance i by chunk_size.
    - Otherwise, write single char, flush, sleep _char_delay(ch), advance i by 1.
    - After loop: write '\n' and flush.
    - Use sys.stdout.write + sys.stdout.flush (NOT print, NOT Rich Console).

    `print_reasoning_preamble() -> None`:
    - Create Console(markup=True, highlight=False).
    - Print `"[italic dim]*Analyzing the request and preparing response...*[/italic dim]\n"` per D-04.

    `print_chapter_header(chapter_num: int, title: str) -> None`:
    - Create Console(markup=True, highlight=False).
    - Print `f"\n[bold]第{chapter_num}章  {title}[/bold]\n"` per D-12/DISP-06.

    `print_nav_hints(chapter_index: int, total: int) -> None`:
    - Create Console(markup=True, highlight=False).
    - Build hints list:
      - If chapter_index < total - 1: add "[dim]/novel next[/dim] -- 下一章"
      - If chapter_index > 0: add "[dim]/novel prev[/dim] -- 上一章"
      - Always add "[dim]/novel toc[/dim] -- 目录"
    - Print newline then "  " + "   |   ".join(hints) per D-12/DISP-07.

    `stream_chapter(chapter_num: int, title: str, content: str, chapter_index: int, total: int) -> None`:
    - Calls print_reasoning_preamble(), print_chapter_header(chapter_num, title), stream_text(content), print_nav_hints(chapter_index, total) in sequence.
    - This is the main entry point that Plan 03 will wire into commands.py.

    **Module-level console:** Do NOT create a module-level Console — create it inside each function that needs it, to avoid Rich detecting test environments incorrectly.
